# File_Filter


## Флаги:
| Флаги          | Описание                                       |
|----------------|------------------------------------------------|
| -o             | Задать путь к месту сохранения файлов          |
| -p             | Добавить префикс к названию сохраняемых файлов |
| -a             | Добавление в существующие файлы                |
| -s             | Краткая статистика                             |                            
| -f             | Полная статистика                              |

### Пример: ```-o /java/test/ -p prefix_ -a -f text.txt tst2.txt```
